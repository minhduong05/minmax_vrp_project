# Algorithm comparison experiments

Dung folder nay de luu thuc nghiem so sanh cac thuat toan tren cac bo data khac nhau.

Template CSV: `../templates/algorithm_comparison.csv`.

Moi dong nen tuong ung voi mot lan chay `(algorithm, instance, seed, time_limit)`.

Khi so sanh nen giu cong bang:

- cung instance;
- cung `k`;
- cung `time_limit`;
- cung che do return-to-depot;
- chay nhieu seed voi thuat toan co random.
